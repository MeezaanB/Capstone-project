ROUTES FOR NIGHTCRAWLER

HOME :
Method Route Definition
GET / View home.
USER :
Methods Routes Definition
GET /users

/user/:id 		View all users.

View single user by ID.
POST /login

/register 		Log In as existing user.

Allows new user to register.
PUT /user/:id Update a user by ID
DELETE /user/:id Delete a user by ID
PRODUCTS :
Methods Routes Definition
GET /products

/product/:id 		View all products.

View a single product.
POST /product Create a new product.
PUT /product/:id Update a product by ID
DELETE /product/:id Delete a product by ID
CART :
Methods Routes Definition
GET /user/:id/carts View what is inside a single user's cart
POST /user/:id/cart Adds item to cart
PUT /user/:id/cart Updates item on the cart
DELETE /user/:id/cart/:id
/user/:id/cart 		Removes single item from cart by productID
Removes everything from the cart by userID
